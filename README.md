# Golden Opportunity Trading & Contracting W.L.L.

Welcome to the official website repository of **Golden Opportunity Trading & Contracting W.L.L.**, a leading Qatari company specializing in construction, architectural design, facility operations, and maintenance since 2008.

---

## About

Golden Opportunity has built a strong reputation for quality, reliability, and innovation in residential, commercial, and governmental projects throughout Qatar. Our commitment to precision, professionalism, and client satisfaction drives every project we undertake.

---

## Website Structure

This repository contains the following pages:

- **Home** (`index.html`): Introduction and overview of the company.
- **About Us** (`about.html`): Detailed company background, growth, and vision.
- **Contact Us** (`contact.html`): Contact form and company contact details.

---

## Technologies Used

- HTML5 for structure and content.
- CSS3 for styling and responsive design.
- Google Fonts (Poppins) for modern typography.

## How to Use

Clone or download this repository to view or modify the website locally. Open `index.html` in any modern web browser to see the homepage. Use the navigation links to access other pages.
